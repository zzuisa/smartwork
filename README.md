# smartwork

#### ✏ 功能描述:

1. 自动生成每日工作文件夹
2. 基于工作日志导出**月底交付件报表** `V 1.4.1`

pandas==1.4.4
et-xmlfile==1.1.0
numpy==1.23.3

#### ⚙ Run:

1. 统计交付件：`python3 ./main.py`
2. 生成每日工作目录：`python3 ./main.py auto`
