# smartwork

#### ✏ 功能描述:

基于工作日志导出`月底交付件报表` **V 1.0**



#### 📌更新明细：

| 版本号    | 修改内容                                                     |
| --------- | ------------------------------------------------------------ |
| `V 1.0`   | 项目初始化                                                   |
| `V 1.0.1` | 添加手动调整指定月份，修改Readme.md，修复bug：无法存储多月份的统计信息 |



#### 📋 Requirements:

> python==3.8.8

openpyxl==3.0.10
pandas==1.4.4
et-xmlfile==1.1.0
numpy==1.23.3
tqdm==4.64.1
